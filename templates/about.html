About this project

Making sign language accessible to everyone.

SignSpeak is a final-year project that bridges the communication gap between the hearing and deaf communities using real-time AI-powered hand gesture recognition.

What this project does

Using your device camera and a trained neural network model, SignSpeak detects ASL hand signs (A–Z) in real time. Letters are accumulated into words, words into sentences, and the final caption can be read aloud using text-to-voice synthesis.

The system uses a 2-second hold confirmation to prevent accidental letter detection — only intentional, stable signs get added to the caption. Word suggestions appear as you sign to speed up communication.

Everything runs locally in the browser connected to a Django backend — no external cloud services required for detection.

Technology stack

Django Web framework
OpenCV Webcam + image processing
cvzone Hand tracking
TensorFlow / Keras Neural network inference
NLTK Brown Corpus Frequency-ranked suggestions
gTTS Text to speech

How the model was trained

The classification model was trained using Google Teachable Machine on a custom dataset of hand sign images for A–Z. Each letter was captured with ~200 images under varying lighting and hand positions.

The exported Keras model runs at inference time inside the Django streaming view, processing each webcam frame with cvzone's hand detector to crop and normalise the hand region before classification.

Why we built this

Sign language is used by over 70 million deaf people worldwide, yet most people cannot understand it. Real-time translation tools can reduce this barrier significantly.

This project demonstrates that accessible AI tools can be built with open-source technology, a webcam, and a trained model — no expensive hardware or cloud subscriptions required.
